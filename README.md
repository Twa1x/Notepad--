# Name
Notepad--

# Description
- The application is a clone of Notepad++
- Functions :
    - New, Open, Save, Save as
    - Find, Replace, Replace all

# Technologies used
- .NET(C#, WPF) 
-  Model–view–viewmodel (MVVM) architecture
